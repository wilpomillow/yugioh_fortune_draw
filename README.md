# Yu-Gi-Oh Fortune Draw

## Setup
1) Put your font file here:
- `public/fonts/MatrixRegularSmallCaps.woff2`

2) Install deps:
```bash
npm install
```

3) Run:
```bash
npm run dev
```

## Notes
- Daily lock is stored in `localStorage` (device-based).
- Remote images are allowed via `next.config.js`.
